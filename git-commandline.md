## Using git
### clone repositoy
git clone git@github.com:username/reponame.git

### add file to repositoy
git add xxx
git commit -m 'comment'
------
or
git commit -a -m 'comment'

### git status
git status

### git diff
git diff
git diff --cached

### delete file
rm README
git rm README
git commit -m 'commennt'

### push
git push [remote] [branch]
